# Diwali Sales Analysis

## Project Overview

This project analyzes customer purchasing behavior during Diwali sales using Python.

## Objectives

- Understand customer demographics
- Identify high revenue segments
- Analyze geographical sales performance
- Generate business recommendations

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights

- Females contribute majority of sales.
- Age group 26-35 dominates purchases.
- Uttar Pradesh generates highest revenue.
- Food, Clothing and Electronics are top categories.

## Business Recommendations

- Target female customers.
- Focus campaigns in high-performing states.
- Improve festive inventory planning.
